Class: GraphQL::Directive < Object Directives are server-defined hooks for modifying execution. Two directives are included out-of-the-box: - @skip(if: ...) Skips the tagged field if the value of if is true - @include(if: ...) Includes the tagged field only if if is true Includes: GraphQL::Define::InstanceDefinable Instance methods: default_directive?, initialize, inspect, on_field?, on_fragment?, on_operation?, to_s, type_class
